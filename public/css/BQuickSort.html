Quick sort

Quick sort is a highly efficient sorting algorithm and is based on partitioning of array of data into smaller arrays. A large array is partitioned into two arrays one of which holds values smaller than the specified value, say pivot, based on which the partition is made and another array holds values greater than the pivot value.
Quicksort partitions an array and then calls itself recursively twice to sort the two resulting subarrays. This algorithm is quite efficient for large-sized data sets as its average and worst-case complexity are O(n2), respectively.

Quick sort Algorithm

Quick sort is one of the divide and conquer algorithm. In quick sort we select a pivot element randomly and then partition the array into two halves and sort them. There are different ways by which we can select the pivot element from the array. They are:

  • First element as pivot
  • Last element as pivot
  • Mid element as pivot
  • Any random element as pivot

    • The main key process of quick sort is partitioning. In the given array, select element x as pivot. Putting that pivot element x at its correct position arrange all elements smaller than x to the left of pivot element x and all the greater elements to its right. This is the idea to sort in quick sort. 

    
			/* low  --> Starting index,  high  --> Ending index */
	  quickSort(arr[], low, high)
	  {
		  if (low < high)
		  {
			  /* pi is partitioning index, arr[pi] is now
				 at right place */
			  pi = partition(arr, low, high);
	  
			  quickSort(arr, low, pi - 1);  // Before pi
			  quickSort(arr, pi + 1, high); // After pi
		  }
	  }

    Quick sort Example

    Quick sort is said to be more efficient than merge sort. In merge sort we divided the array in two subarrays, sort those subarrays and merge them. But in quick sort we don’t really need to divide the array and sort and merge again. In quick sort an array a[1:n] where p is first element and q is last element of array and j is mid element. Consider these two arrays that is (p, j-1) and (j+1, q) and rearrange them such that p < q always. At the end we don’t need to combine or merge arrays as they were never really divided and we get a sorted array with ascending order. Quick sort is a space-optimised version of Binary tree sort. 

    
		// Quick sort in C++
	
		#include 
		using namespace std;
		
		// function to swap elements
		void swap(int *a, int *b) {
		  int t = *a;
		  *a = *b;
		  *b = t;
		}
		
		// function to print the array
		void printArray(int array[], int size) {
		  int i;
		  for (i = 0; i < size; i++)
			cout << array[i] << " ";
		  cout << endl;
		}
		
		// function to rearrange array (find the partition point)
		int partition(int array[], int low, int high) {
			
		  // select the rightmost element as pivot
		  int pivot = array[high];
		  
		  // pointer for greater element
		  int i = (low - 1);
		
		  // traverse each element of the array
		  // compare them with the pivot
		  for (int j = low; j < high; j++) {
			if (array[j] <= pivot) {
				
			  // if element smaller than pivot is found
			  // swap it with the greater element pointed by i
			  i++;
			  
			  // swap element at i with element at j
			  swap(&array[i], &array[j]);
			}
		  }
		  
		  // swap pivot with the greater element at i
		  swap(&array[i + 1], &array[high]);
		  
		  // return the partition point
		  return (i + 1);
		}
		
		void quickSort(int array[], int low, int high) {
		  if (low < high) {
			  
			// find the pivot element such that
			// elements smaller than pivot are on left of pivot
			// elements greater than pivot are on righ of pivot
			int pi = partition(array, low, high);
		
			// recursive call on the left of pivot
			quickSort(array, low, pi - 1);
		
			// recursive call on the right of pivot
			quickSort(array, pi + 1, high);
		  }
		}
		
		// Driver code
		int main() {
		  int data[] = {8, 7, 6, 1, 0, 9, 2};
		  int n = sizeof(data) / sizeof(data[0]);
		  
		  cout << "Unsorted Array: \n";
		  printArray(data, n);
		  
		  // perform quicksort on data
		  quickSort(data, 0, n - 1);
		  
		  cout << "Sorted array in ascending order: \n";
		  printArray(data, n);
		}

    Applications:
    Now you might think how this algorithm can be used in real world. Quick sort is the fastest method to solve any sorting problems. Some examples are:

  • Commercial computing – From list of accounts sorting data through their names, id, dates etc.
  • Numerical computing – Sorting numeric data with the help of quick sort
  • Searching information – From a given list of information quick sort is best fastest way for searching.

    • Complexity Analysis:

    Time Complexity– Time taken by quick sort can be written as
    T(n) = T(k) + T(n-k-1) + Ꝋ(n)

  • Worst case – Worst case occurs in two situations. First if we pick the smallest or greatest element from the array. And second if the array is already sorted in ascending or descending order.
    Recurrence of worst case is - T(n) = T(n-1) + Ꝋ(n)
    Solution - Ꝋ(n2)


  • Best case – Best case occurs when we select the middle element as pivot.
    Recurrence of best case is – T(n) = 2T(n/2) + Ꝋ(n)
    Solution - Ꝋ(nLogn)


  • Average case - Recurrence of average case – T(n) = T(n/9) + T(9n/10) + Ꝋ(n)
    Solution – 0(nLogn)


  • Space complexity – space complexity depends on the space used in recursion stack.
    Worst case – 0(n)
    Average case – 0(Log n)
