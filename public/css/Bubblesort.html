Bubble sort

Bubble sort is a simple sorting algorithm. In this technique you compare two adjacent elements and swap them to place them in order. Thus, bubble sort is also known as comparison sort algorithm.

Bubble sort Algorithm


			bubbleSort( Arr[], totat_elements)
   
   for i = 0 to total_elements - 1 do:
      swapped = false
		
      for j = 0 to total_elements - i - 2 do:
      
         /* compare the adjacent elements */   
         if Arr[j] > Arr[j+1] then
            /* swap them */
            swap(Arr[j], Arr[j+1])		 
            swapped = true
         end if
         
      end for
      
      /*if no number was swapped that means 
      array is sorted now, break the loop.*/
      
      if(not swapped) then
         break
      end if
      
   end for
   
end

The algorithm is quite simple. Assume a list or array of n elements and the swap function compares and if the first element is greater than the second element then swap them. This loop goes on till the last element and the array is sorted. But this algorithm is not useful for sorting huge data. So we can apply this sorting algorithm in few cases where data is small. But still learning the bubble sort is important as it teaches the basic sorting technique. Lets try to understand with a small example

As you can see in the above figure, two elements are compared and swapped and this process continues till all the elements are sorted and the finally we get a sorted array.

Pseudo code for Bubble sort:
begin BubbleSort(list)

		// Bubble sort in C++

		#include <iostream>
		using namespace std;
		
		// perform bubble sort
		void bubbleSort(int array[], int size) {
		
		  // loop to access each array element
		  for (int step = 0; step < (size-1); ++step) {
			  
			// loop to compare array elements
			for (int i = 0; i < size - (step-1); ++i) {
		
			  // compare two adjacent elements
			  // change > to < to sort in descending order
			  if (array[i] > array[i + 1]) {
		
				// swapping elements if elements
				// are not in the intended order
				int temp = array[i];
				array[i] = array[i + 1];
				array[i + 1] = temp;
			  }
			}
		  }
		}
		
		// print array
		void printArray(int array[], int size) {
		  for (int i = 0; i < size; ++i) {
			cout << "  " << array[i];
		  }
		  cout << "\n";
		}
		
		int main() {
		  int data[] = {-2, 45, 0, 11, -9};
		  cout<<"Umsorted array is:::  ";
		  for (int i = 0; i < 5; ++i) {
			cout << "  " << data[i];
		  }
		  
		  // find array's length
		  int size = sizeof(data) / sizeof(data[0]);
		  
		  bubbleSort(data, size);
		  
		  cout << "\nSorted Array in Ascending Order:\n";  
		  printArray(data, size);
		}

Applications:
As bubble sort cannot be used to sort huge data, it is mostly used for educational purpose to teach students the basic concept of sorting.
If we think about the real world problem consider a queue of people standing at bus stand and trying to arrange them according to their height.

Complexity Analysis:

Time Complexity

  • Best case – Best case time complexity occurs when the array is already sorted, then no swapping is required. So, it is 0(n) as it has to traverse all elements only once.

  • Worst case and Average case – Both are same for bubble sort. It occurs when array is in reversed order i.e. n2 or 0(n2).


  • Space complexity - Space complexity for bubble sort is 0(1), because it needs just 1 temporary memory space for swapping.
